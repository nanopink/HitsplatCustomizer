
# Hitsplat Customizer

Make combat readable when the damage starts flying.

Hitsplat Customizer redraws OSRS hitsplats with the game's own sprites while giving you control over how many appear, where they land, and how long they stay visible.

The default `RuneScape` preset keeps the familiar four-corner feel, including zero hitsplats, so the game still reads like OSRS.

<img width="602" height="370" alt="Recording 2026-07-09 at 08 00 18" src="https://github.com/user-attachments/assets/2340c3e9-fcc3-4732-8249-927d6500e339" />

Use `Ruined Heir's 1 tick` when you want quick, clean burst feedback. Its animation lifetime is tuned under a tick, making real-time damage checks easier without splats piling up.

<img width="779" height="527" alt="Recording 2026-07-09 at 08 02 15" src="https://github.com/user-attachments/assets/98b2c87b-f391-4f26-800d-1f3ba9945df6" />

Shape the layout around how you actually play. Tune shape, direction, behavior, radius, spacing, offsets, opacity, and fade timing.

<img width="721" height="533" alt="Recording 2026-07-09 at 04 41 37" src="https://github.com/user-attachments/assets/a784a405-6b59-4039-a8f6-e375bf637eb8" />

Keep it focused with source filters, prioritize your own hits when space is limited, or disable hitsplats by target category when you want a cleaner screen.
